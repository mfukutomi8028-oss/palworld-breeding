# パルワールド配合記録ボード v7

GitHub Pagesで公開できる、Firebase Realtime Database対応の配合記録ボードです。

## v7 修正内容

- 左上・メニューのタマゴ画像を、ユーザー指定の「平凡なタマゴ」画像に差し替えました。
- ブラウザタブのfaviconを、ユーザー指定の公式風パル画像に設定しました。
- ネタバレ防止のため、未登録パルの画像は「？」画像で表示するようにしました。
- 一度でも配合記録に追加されたパルだけ、一覧・詳細・入力候補・絞り込み候補で実画像が表示されます。
- パル名はこれまで通り表示されます。画像のみ伏せます。

## GitHub Pagesへの反映

zipを解凍し、中身をリポジトリ直下へ上書きアップロードしてください。

- index.html
- style.css
- app.js
- config.js
- config.example.js
- firebase-rules.json
- README.md
- assets フォルダ

反映後、公開ページで `Ctrl + F5` を押してください。
古い表示が残る場合は、URL末尾に `?v=7` を付けて確認してください。

## Firebaseについて

`config.js` にFirebase設定を入れると共同編集ONになります。
未設定の場合はブラウザ内のローカル保存で動きます。


## v8 修正内容

- フブキジカの画像が表示されない問題を修正しました。
- 未登録パルの画像は、追加フォーム・絞り込み候補・一覧で「？」画像になるよう再調整しました。
- 一度でも配合記録に登場したパルのみ画像を表示します。
- 登録画面の「確認状況」入力を廃止しました。
- 結果パルが入力されている場合は自動で「配合確認済み」、未入力の場合は自動で「確認中」になります。
- タブアイコンを `favicon-v8.png` に変更し、ブラウザキャッシュを避けるためURLに `?v=8` を付けています。
- 平凡なタマゴ画像を添付画像から切り抜き、`assets/plain-egg.png` として同梱しました。


## v9 修正内容

- パル名検索・親A/親B/結果パルの候補検索で、ひらがな入力に対応しました。
  - 例：`ふぶきじか` → `フブキジカ`
  - 例：`もころん` → `モコロン`
  - 例：`らべろっと` → `ラベロット`
- 入力したひらがな名を保存する場合も、パル一覧に存在する名前はカタカナ名へ正規化します。
- ブラウザ側の古いパル一覧キャッシュを避けるため、パルデータキャッシュキーをv9へ更新しました。
